This is a boilerplate for a hugo static site, using gulp as a build tool, Sass
and PostCSS for CSS processing, and Turretcss as utility library.

## Prerequisites

- Hugo
- Node.js
- npm

## Install

Clone repository

````bash
npm install
````

## Compile assets

````bash
gulp
````

## Development

````bash
hugo server
````

## Production

````bash
hugo
````
